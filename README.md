# 🍕 RusticOrder - User Registration

Multi-platform application designed for the **Pizza Rústica Gourmet** chain. Allows customers to easily register to access the online ordering service.

## 🚀 Key Features

- **Cross-Platform Architecture:** Built with **Flutter**, designed to compile natively on Android, iOS, and Web from a single codebase.
- **Customer Registration:** Complete form with data validation (Name, Email, Address, Password).
- **Thematic Design:** Custom interface with brand visual identity (Logo, corporate colors, checkered tablecloth pattern).
- **Intuitive Navigation:** Clear user flow between Home, Registration, and Login screens.

## 🛠️ Technologies Used

- **Language:** Dart
- **Framework:** Flutter
- **Key Widgets:** `TextField`, `ElevatedButton`, `Column`, `Container`, `ThemeData`.
- **Focus:** UI/UX centered on the gastronomic user experience.

## ⚙️ How to Run the Project

If you wish to test this code in your local environment:

1.  Make sure you have [Flutter installed](https://flutter.dev/docs/get-started/install).
2.  Clone this repository:
    ```bash
    git clone https://github.com/sergiomoscoso/RusticOrder-Staff.git
    ```
3.  Install dependencies:
    ```bash
    flutter pub get
    ```
4.  Run the application:
    ```bash
    flutter run
    ```
